## Deskripsi
Warehouse.in adalah aplikasi berbasis web untuk mengelola data gudang.  
Aplikasi ini menggunakan PHP dan database sebagai media penyimpanan data.

## Panduan Instalasi dan Penggunaan

### Langkah Instalasi
1. Pull repository Warehouse.in dari GitHub ke komputer Kamu.  
2. Pindahkan folder aplikasi ke direktori lokal yang dapat diakses oleh PHP. Contoh htdocs pada XAMPP.  
3. Siapkan database menggunakan MySQL atau sistem database sejenis.  
4. Masukkan data awal aplikasi ke dalam database yang telah dibuat.  
5. Atur konfigurasi koneksi database pada file aplikasi.  
6. Aktifkan server pendukung seperti Apache dan MySQL.  
7. Jalankan server sesuai dengan konfigurasi yang telah disiapkan.  
8. Akses aplikasi melalui web browser menggunakan alamat server lokal. Contoh http://localhost/Warehouse.in  

### Cara Penggunaan
1. Pastikan server dan database sudah berjalan.  
2. Buka aplikasi melalui web browser.  
3. Gunakan fitur aplikasi untuk mengelola data gudang.
